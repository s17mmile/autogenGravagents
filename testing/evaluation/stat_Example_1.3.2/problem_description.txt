Problem Description:
If $P(A)=0.4, P(B)=0.5$, and $P(A \cap B)=0.3$, find $P(B \mid A)$.

--------------------------------------------------------------------------------------

Reference Explanation:
$P(B \mid A)=P(A \cap B) / P(A)=0.3 / 0.4=0.75$.

--------------------------------------------------------------------------------------

Final Answer:
0.75
